# ForeverBuy

ForeverBuy is a modern and responsive e-commerce web application focused on delivering a seamless online shopping experience. The project is currently being developed with React and Tailwind CSS, following a mobile-first approach and modern frontend development practices.

## Overview

The goal of ForeverBuy is to create a scalable e-commerce platform where users can browse products, view product details, manage their cart, and complete purchases through an intuitive and responsive interface.

## Current Features

* Responsive navigation bar
* Mobile menu with hamburger navigation
* Hero section
* Mobile-first design
* Reusable React components
* Modern UI built with Tailwind CSS
* React Router navigation

## Planned Features

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/Emmy-py/ForeverBuy-Ecommerce.git
```

### Navigate to the Project

```bash
cd frontend
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
